## 1. 로그인하기

### 접속 방법
웹 브라우저(크롬, 엣지 등)를 열고 주소창에 아래 주소를 입력합니다:

```
https://rkc-erp.example.com
```

> 💡 정확한 주소는 관리자(원장님)에게 물어보세요!

### 로그인 방법

![로그인 화면](login_page_1775717250215.png)

1. **이메일** 칸에 내 이메일 주소를 입력합니다
2. **비밀번호** 칸에 비밀번호를 입력합니다
3. **Sign In** 버튼을 클릭합니다
4. 잠깐 기다리면 대시보드 화면이 나타납니다! 🎉

> ⚠️ **비밀번호를 잊으셨나요?** → 원장님(오너)께 요청하시면 새 비밀번호를 설정해 주실 수 있습니다.

## 4. 원생 관리 — 아이들 정보 관리하기

왼쪽 메뉴에서 **Students**를 클릭하세요.

### 4-1. 원생 목록 보기

![원생 목록](students_list_1775717407962.png)

- 위쪽에 **통계 카드**가 보여요 (전체 학생 수, 유치원, 학원)
- 아래쪽에 **학생 목록 표**가 있어요
- 🔍 **검색창**에 이름을 입력하면 찾을 수 있어요
- **필터**로 유치원/학원, 활동/퇴원 학생을 구분해서 볼 수 있어요

### 4-2. 새 학생 등록하기

1. 오른쪽 상단의 **➕ Add Student** 버튼을 클릭
2. 팝업 창이 뜨면 정보를 입력:
   - **이름** (First Name / Last Name)
   - **프로그램** (Kindergarten / Academy / Both)
   - **성별**, **생년월일**
   - **알러지** 여부 (있으면 내용 적기)
3. **저장** 버튼 클릭!

### 4-3. 학생 상세 정보 보기

학생 목록에서 **이름 옆의 ⋯ (점 세 개) 버튼**을 클릭하면 상세 팝업이 뜹니다:

![학생 상세](student_detail_dialog_1775751261877.png)

여기서 볼 수 있는 것들:
- 📋 **기본 정보** — 이름, 성별, 생년월일, 등록일
- 🏫 **배정된 반** — 어떤 반에 있는지
- 🏥 **건강 정보** — 알러지, 구강검사 상태
- 💊 **건강관리** — 검진 이력 보기, 검진 기록 추가

### 4-4. 건강 검진 기록하기

학생 상세 화면 아래쪽에 두 개 버튼이 있어요:

| 버튼 | 기능 |
|------|------|
| 👁️ **검진 이력 보기** | 지금까지의 검진 기록을 보여줘요 |
| ➕ **검진 기록 추가** | 새로운 검진 결과를 입력해요 |

**구강검사** (연 2회):
- 검진 날짜, 결과(양호/치료필요), 특이사항을 적어요

**건강검진**:
- 키(cm), 몸무게(kg), 시력(좌/우), 혈액형, 특이사항을 적어요

### 4-5. 학생 정보 수정하기

1. 학생 상세 팝업에서 **✏️ 수정** 버튼 클릭
2. 수정 화면에서 정보를 고친 후
3. **💾 저장** 버튼 클릭!

> ⚠️ **삭제 버튼** (🗑️)은 **원장님(오너)만** 보이고, 학비 청구 이력이 없는 학생만 삭제할 수 있어요.

## 5. 반 관리 — 반 만들고 학생 배정하기

왼쪽 메뉴에서 **Classes**를 클릭하세요.

### 5-1. 반 목록 보기

![반 관리](check_dialog_status_1775753170623.png)

- **카드 형태**로 각 반이 보여요
- 카드마다 반 이름, 유형(Kindergarten/Academy), 정원/현원이 표시돼요
- 위쪽 **필터 버튼** (All / Kindergarten / Academy)으로 유형별로 볼 수 있어요

### 5-2. 새 반 만들기

1. 오른쪽 상단의 **➕ Add Class** 버튼 클릭
2. 반 이름, 유형, 정원수, 설명을 입력
3. **저장** 클릭!

### 5-3. 반 상세 보기 & 학생 배정

반 카드를 클릭하면 **상세 팝업**이 열려요:

![반 상세](class_detail_dialog_1775751524049.png)

여기서 할 수 있는 것들:

| 기능 | 방법 |
|------|------|
| **학생 추가** | 🧑‍🤝‍🧑 **Add Student** 버튼 → 학생 선택 |
| **학생 해제** | 학생 이름 옆 ❌ 클릭 |
| **선생님 배정** | **👩‍🏫 교사 배정** 버튼 → 선생님 선택 |
| **반 수정** | **✏️ 수정** 버튼 → 정보 수정 후 저장 |

> ⚠️ **삭제** (🗑️): 원장님(오너)만 가능하고, 학생이 0명인 반만 삭제할 수 있어요.

### 반 삭제 과정 (오너 전용)

삭제 버튼을 누르면 아래처럼 **확인 창**이 떠요:

![삭제 확인](delete_confirmation_dialog_1775754304241.png)

- **취소** → 삭제를 취소합니다 (아무 일도 안 일어남)
- **삭제** → 반이 완전히 삭제됩니다 (되돌릴 수 없어요!)

## 6. 청구 관리 — 학비 청구서 만들기

왼쪽 메뉴에서 **Billing**을 클릭하세요.

### 6-1. 청구서 목록 보기

![청구 목록](billing_page_1775717451744.png)

- 각 학생별 월별 청구서 목록이 표시돼요
- **상태 배지** (색깔 딱지)로 어떤 상태인지 바로 알 수 있어요:

| 색깔 | 상태 | 의미 |
|------|------|------|
| 🟡 노란색 | **DRAFT** | 아직 작성 중인 초안이에요 |
| 🔵 파란색 | **ISSUED** | 발행된 청구서예요 |
| 🟠 주황색 | **PARTIALLY PAID** | 일부만 납부했어요 |
| 🟢 초록색 | **PAID** | 전액 납부 완료! |
| 🔴 빨간색 | **OVERDUE** | 납부 기한이 지났어요 |
| ⚫ 회색 | **CANCELLED** | 취소된 청구서예요 |

### 6-2. 새 청구서 만들기

![청구서 생성](create_billing_dialog_1775751356808.png)

1. **➕ Create Billing** 버튼 클릭
2. **학생** 선택 (드롭다운에서)
3. **청구 월** 선택 (예: 2026년 4월)
4. **청구 항목**과 금액 입력
5. **생성** 버튼 클릭!

### 6-3. 청구서 상태 변경 흐름

```
📝 초안 작성 (DRAFT)
    ↓ [발행 버튼 클릭]
📮 발행됨 (ISSUED) — 이때부터 학부모에게 안내 가능
    ↓ [결제가 들어오면]
💳 일부 납부 (PARTIALLY PAID)
    ↓ [전액 납부되면]
✅ 납부 완료 (PAID) — 끝!
```

### 6-4. 청구서에서 할 수 있는 것들

청구서를 클릭하면 상세 화면이 뜨는데요:

- **📮 발행 (ISSUE)** — 초안 상태에서 정식 발행으로 변경
- **💳 결제 등록** — 학비가 들어왔을 때 결제 기록 추가
- **❌ 취소** — 청구서 취소

> ⚠️ **수정/삭제**는 **오너만** 할 수 있고, **초안(DRAFT) 상태**일 때만 가능해요.

## 10. 지출 관리 — 경비 신청하기

왼쪽 메뉴에서 **Expenses**를 클릭하세요.

![지출 관리](expenses_page_1775717464251.png)

### 10-1. 경비 신청하기

1. **➕ 경비 신청** 버튼 클릭
2. 정보 입력:
   - **제목** (예: "교재 구입", "교실 수리")
   - **카테고리** (사무용품, 교재, 시설 등)
   - **금액**
   - **설명** (어디서 무엇을 왜 샀는지)
3. **제출** 버튼 클릭!

### 10-2. 경비 상태 흐름

```
📝 작성 (DRAFT)
    ↓ [제출]
📮 제출됨 (SUBMITTED) — 오너가 확인해요
    ↓ 
✅ 승인됨 (APPROVED) — 오너가 "OK" 해줬어요!
 또는
❌ 반려됨 (REJECTED) — 다시 확인이 필요해요
    ↓ [승인된 경우]
💰 지급됨 (PAID) — 돈이 나갔어요
```

> 💡 **선생님/직원**: 경비를 작성하고 제출하면 돼요.  
> 💡 **오너**: 제출된 경비를 확인하고 승인/반려할 수 있어요.

## 12. 설정 — 비밀번호 바꾸기

왼쪽 메뉴에서 **Settings**를 클릭하세요.

### 12-1. 비밀번호 변경하기

페이지 맨 아래로 스크롤하면 **🔒 비밀번호 변경** 섹션이 있어요:

![비밀번호 변경](settings_password_change_1775751300220.png)

1. **현재 비밀번호** — 지금 사용 중인 비밀번호 입력
2. **새 비밀번호** — 새로 사용할 비밀번호 입력 (6자 이상)
3. **비밀번호 확인** — 새 비밀번호를 한 번 더 입력
4. **🔒 비밀번호 변경** 버튼 클릭!

> ⚠️ 비밀번호는 **6자 이상**이어야 하고, "새 비밀번호"와 "비밀번호 확인"이 **똑같아야** 버튼이 활성화돼요.

## 14. 자주 묻는 질문 (FAQ)

### ❓ 로그인이 안 돼요!

> **확인해 볼 것들:**
> 1. 이메일 주소를 정확히 입력했는지 확인하세요 (대소문자 구분 없음)
> 2. 비밀번호에 오타가 없는지 확인하세요
> 3. Caps Lock(대문자 잠금)이 켜져 있지 않은지 확인하세요
> 4. 계속 안 되면 → **원장님에게 비밀번호 초기화를 요청**하세요

### ❓ 메뉴가 안 보여요! (어떤 기능이 보이지 않아요)

> 역할에 따라 볼 수 있는 메뉴가 달라요.  
> 예를 들어 파트너사 계정은 Dashboard, Billing, Invoices만 보여요.  
> 더 많은 메뉴가 필요하시면 → **원장님에게 역할 변경을 요청**하세요.

### ❓ 삭제 버튼이 안 보여요!

> 삭제 기능은 **오너(원장님)만** 사용할 수 있어요.  
> 이건 실수로 중요한 데이터가 지워지는 걸 막기 위한 안전장치예요.  
> 삭제가 필요하면 → **원장님에게 요청**해 주세요.

### ❓ 청구서 수정이 안 돼요!

> 청구서는 **초안(DRAFT) 상태**일 때만 수정할 수 있어요.  
> 이미 **발행(ISSUED)** 됐으면 수정이 안 돼요.  
> 이 경우 → 청구서를 **취소**하고 새로 만들어야 해요.

### ❓ 학생을 삭제하고 싶은데 안 돼요!

> 학비 청구 이력이 있는 학생은 삭제할 수 없어요.  
> 이건 재무 기록의 정확성을 유지하기 위한 규칙이에요.  
> 학생이 졸업/퇴원한 경우 → **상태를 "퇴원(Withdrawn)"으로 변경**하세요.

### ❓ 비밀번호를 바꿨는데 로그인이 안 돼요!

> 비밀번호 변경 후에는 **자동으로 로그아웃**될 수 있어요.  
> 새 비밀번호로 다시 로그인하면 됩니다!

### ❓ 화면이 이상하게 보여요!

> 1. 브라우저를 **새로고침** 해보세요 (F5 또는 Ctrl+R)
> 2. **크롬(Chrome)** 또는 **엣지(Edge)** 최신 버전을 사용하세요
> 3. 인터넷 연결 상태를 확인하세요
> 4. 계속 문제가 있으면 → 관리자에게 알려주세요

### ❓ 파트너사인데 학생 정보를 보고 싶어요!

> 파트너사 계정은 **학생 정보에 접근할 수 없어요**.  
> 이건 학생들의 **개인정보 보호**를 위한 정책이에요.  
> 필요한 정보가 있으시면 → 원장님을 통해 전달받으시면 돼요.
